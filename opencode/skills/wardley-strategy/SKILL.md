# Wardley Mapping — Strategy

## Purpose

Derive actionable insights from the map.

## Core Rules

- TIE everything to the map
- BE specific
- AVOID generic advice

## Interaction

Ask:

- Where should we invest?
- What should we standardize or outsource?
- What is likely to change?
- Where are we over-engineering?

## Output

Provide:

- Key observations
- Risks or tensions
- Suggested focus areas

## Constraints

- NO generic best practices
- NO theory
- MUST reference earlier reasoning

## Exit Criteria

- Clear, actionable insights

## Forbidden

- Rebuilding earlier steps
